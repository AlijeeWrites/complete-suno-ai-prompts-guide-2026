## Optimal Style Formula for Suno v5.5

Use this winning structure:
[Genre/Subgenre], [Tempo/Energy/BPM], [Key instruments], [Vocal style], [Production/Mood], [Modifiers]


### Best Practices

- Keep it between **4 to 8 descriptors**
- Put **genre and vocal type early**
- Use negatives sparingly (`-glossy`, `-autotune`, `-hiss`)
- **Powerful modifiers:** `[Hyper-Realistic]`, `[No Vocals]`, `[Instrumental]`, `[Sustained]`
